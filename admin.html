Danger Zone

การกระทำเหล่านี้ไม่สามารถกู้คืนได้ กรุณาใช้ด้วยความระมัดระวัง

⚠️ คำเตือน: พื้นที่อันตรายสูง

ทุกการกระทำใน Danger Zone จะส่งผลกระทบต่อข้อมูลจริงในระบบ เนื่องจากใช้ localStorage เท่านั้น ข้อมูลที่ถูกลบ ไม่สามารถกู้คืนได้

Factory Reset

ลบข้อมูลทั้งหมดใน localStorage — ผู้ใช้ทุกคน, ความคืบหน้า, สถิติ และ Admin sessions

Clear Admin Sessions

ลบ Admin session ออกจาก localStorage — คุณจะถูก Logout อัตโนมัติ (ข้อมูลผู้ใช้ไม่ถูกกระทบ)

Reset Stage Progress Only

รีเซ็ต Stage progress ของผู้ใช้ทุกคนกลับเป็นค่าเริ่มต้น XP และ Coins ยังคงอยู่

Export User Data
